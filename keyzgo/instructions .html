Инструкции по подключению

Подробные пошаговые руководства по настройке VPN на всех ваших устройствах для безопасного и анонимного серфинга

Android

Инструкция по настройке VPN на устройствах Android

1

Скачайте приложение

Установите официальный клиент OpenVPN Connect из Google Play Store:

Google Play
2

Импортируйте конфигурацию

Откройте приложение → Нажмите "File" → Выберите .ovpn файл из загрузок или облачного хранилища.

3

Подключитесь

Нажмите "Add" → Введите имя профиля → Нажмите "Connect" для установки соединения.

iPhone/iPad

Инструкция по настройке VPN на устройствах iOS

1

Установите приложение

Скачайте OpenVPN Connect из App Store:

App Store
2

Передайте файл конфигурации

Отправьте .ovpn файл себе на email или загрузите в iCloud. Откройте файл через OpenVPN Connect.

3

Настройте подключение

В приложении нажмите "+" → Импортируйте профиль → Разрешите добавление VPN-конфигурации.

4

Активируйте VPN

Выберите созданный профиль и переведите переключатель в положение "ON".

Windows

Инструкция по настройке VPN на компьютерах с Windows

1

Скачайте клиент OpenVPN

Установите официальное приложение с сайта OpenVPN:

Скачать
2

Разместите конфигурационный файл

Скопируйте .ovpn файл в папку: C:\Program Files\OpenVPN\config\

3

Запустите OpenVPN GUI

Откройте программу от имени администратора (правой кнопкой → "Запуск от имени администратора").

4

Подключитесь

Кликните правой кнопкой по иконке в трее → выберите профиль → "Подключиться".

macOS

Инструкция по настройке VPN на компьютерах Apple

1

Установите Tunnelblick

Скачайте и установите бесплатный клиент OpenVPN для Mac:

Скачать
2

Добавьте конфигурацию

Откройте .ovpn файл → Tunnelblick предложит импортировать профиль.

3

Подключитесь

В меню Tunnelblick (иконка в строке меню) выберите "Подключиться".

Smart TV

Инструкция для телевизоров на Android TV

1

Установите приложение

Скачайте APK-файл OpenVPN Connect для Android TV:

APK файл
2

Установка APK

Передайте файл на TV через USB или облако и установите его.

3

Настройте подключение

Откройте приложение → Импортируйте .ovpn файл → Подключитесь.
